css3的功能：CSS3提供了强大而且精准的选择器，提供了多种背景填充方案、可以实现渐变颜色、改变元素形状、角度、加阴影效果，新的报纸布局、弹性盒子，ie6混杂模式的盒子模型，新的计量单位，动画效果等等。 ～一些css3的属性 1。border-radius: border-radius是一个复合属性，它是由border-top-left-radius、border-top-right-radius、border-bottom-left-radius、border-bottom-right-radius四个属性复合而来。 半圆的写法：div { height:50px; width: 100px; border-top-left-radius : 50px; border-top-right-radius: 50px; background-color: red; } 2.box-shadow: box-shadow: x轴偏移量 y轴偏移量 [阴影模糊半径] [阴影扩展半径] [阴影颜色] [投影方式] 3.border-image: border-image: url(xxx.png) number(默认单位是px) repeat / round / stretch 4.rgba 5.background: linear-gradient: linear-gradient([direction], color[percent], color[percent], …); 线性渐变 radial-gradient(shape at position, color[percent], …);径向渐变 6.background-origin background-clip borde-box、padding-box、content-box 7. background-size auto是不改变图片的原始宽高 cover是同比例放大，一张图片填满box contain是尽量让一张图片存进box，多出的部分重复显示 ～伪类选择器 伪类和伪元素之间的区别：伪类是用于给某些选择器添加特殊效果的，相当于我们创建出来一个类，伪元素的效果则是需要添加一个实际元素才能达到的。 CSS3中新增的伪类选择器： 1.:first-child :last-child :nth-child(n) p:nth-child(2n) { background-color: red; } 2.:first-of-type :last-of-type :nth-of-type(n) 3.:not() p:not([class="demo"]) { background-color: red; } 4.:target #demo:target { background-color: green; } click me
5.:enabled 默认:disabled不可用 ～伪元素选择器 1.::selection 这个伪元素可以改变当文本被选中状态下的样式，默认是蓝底白字。 p::selection { color: white; background-color: orange; } 如果我们想设置不允许用户选中文本的话，可以用这个属性： *{ -webkit-user-select: none; } ～图形变换 transform属性 1.rotate 这个属性值是旋转的意思，默认是沿着z轴旋转。单位是deg——度数。 transform: rotateX(45deg) rotateY(45deg) rotateZ(45deg); 2.scale scale的意思是让元素进行缩放，属性值填写的就是缩放倍数，1代表不进行缩放，小于1是缩小，大于1是放大，以元素的中心为缩放原点。 transform: scale(2); 3.skew kew的意思是对元素进行扭曲。 transform: skew(45deg); skewX是沿着X轴进行扭曲，元素的高度不变，左右两个边扭曲一定的角度。 skewY是沿着y轴进行扭曲，元素的宽度不变，上下两个边扭曲一定的角度。 4.translate transform: translateX(-50%) translateY(-50%); 5.translate-origin 这个属性是设置变换原点的作用，上面那些变换的原点默认都是中心点，这个属性就可以更换原点。 transform: rotate(45deg); transform-origin: left top; ～图形过渡 transition就是过渡动画的属性。 这个属性也是一个复合属性，主要包括以下几个子属性： transition-property 指定过渡或者动态模拟的css属性 transition-duration 指定过渡所需要的时间 transition-timing-function 指定过渡函数 transition-delay 指定延迟时间 ～css3动画 二维动画-animation animation是一个复合属性，它包括以下几个组成部分： animation-name：执行动画的keyframes的名字。 animation-duration：执行动画的总时长。 animation-timing-function：指定过渡函数。 animation-delay：执行延迟时间。 animation-direction：动画播放的方式。这个属性的值主要有：normal正常播放模式、reverse倒序播放、alternate动画在奇数次正向播放偶数次倒序播放、alternate-reverse动画在奇数次倒序播放偶数次正向播放。 animation-iteration-count：动画执行的次数。infinite是无限次，写一个数字就是要执行几次。 animation-fill-mode：执行完动画后物体停止的位置。forwards是停在结束的位置上、backwards是快速执行初始帧（这个最好配合延迟来观察）、none是回到初始帧的位置、both是同时具有forwards和backwards的效果。 animation-play-state：控制动画的播放状态。running是播放、paused是暂停。 div:hover { animation-play-state: paused; } div.demo { width: 100px; height: 100px; position: absolute; left: 0px; top: 0px; background-color: red; border-radius: 50%; animation: move 3s ease alternate infinite; } @keyframes move { 0% { top: 0px; } 100% { top: 400px; } } ~三维动画-transform-style 3d动画的一个核心属性就是transform-style了，这个属性可以让我们设置元素是在2d平面还是3d空间，属性值是flat或者preserve-3d。默认情况下是flat（二维）。 除了这个属性，还有另外一个用在父级元素上的属性——perspective景深。
